# ClickUp — Portable All‑in‑One Project Management Platform

## About ClickUp

ClickUp provides a complete project management environment for users who want everything in one place. ClickUp combines tasks, docs, whiteboards, goals, chat, and dashboards. ClickUp supports custom views (list, board, calendar, Gantt, timeline), automations, and 1000+ integrations. ClickUp enables users to manage projects, write documentation, brainstorm on whiteboards, track goals, and chat with team members. ClickUp runs on desktop platforms with offline support. ClickUp is trusted by individuals, startups, enterprises, and remote teams worldwide.
